What is Posey?

Posey is a reading companion for iPhone. Import documents — PDFs, EPUBs, Word files, Markdown, HTML, RTF, and plain text — and read or listen in a clean, focused environment. Take notes anchored to specific passages. Everything stays on your device.

Getting Started

Tap the import button to add a document from Files, iCloud, or any app that supports sharing. Posey extracts the text and opens it in the reader. Tap the play button to start listening. Tap any sentence to jump to that position.

Reading and Listening

Posey highlights each sentence as it's read aloud, keeping your eyes and ears in sync. Two voice modes are available in Preferences. Best Available uses your device's highest-quality voice. Custom lets you choose any voice and adjust the reading speed with a slider.

Notes and Bookmarks

Tap and hold any sentence to create a note or bookmark anchored to that position. All annotations appear in the Notes sheet, sorted by where they occur in the document. Tap any annotation to jump back to the passage.

Table of Contents

For documents with navigable structure — EPUBs, PDFs, Word files, Markdown, and RTF — the TOC button lists all sections. Tap any entry to jump there. The Images tab shows every image in the document as a thumbnail — tap to jump directly to that position.

Search

Find any word or phrase in the current document. Navigate between matches with the arrows.

Images

Images and diagrams appear inline at their correct position in the text. Tap any image to view it full screen. Pinch to zoom.

Troubleshooting

  • Voice not sounding right: Switch between Best Available and Custom in Preferences. Best Available uses the voice you've set in Settings → Accessibility → Spoken Content.
  • Document looks wrong after import: Some PDFs are scanned images. Posey uses on-device OCR to extract text from scanned pages, which may take a moment for long documents.
  • Import not appearing: Make sure you're sharing the file itself, not a link to it.
  • Playback stops unexpectedly: Check that your device isn't set to silence with the Ring/Silent switch. Posey uses your device's audio session and will respect system audio interruptions.
